# 🖥️ Customer Management Web

ระบบหน้าบ้านสำหรับบริหารจัดการข้อมูลลูกค้า พัฒนาด้วย Next.js 15 โดยเน้นประสิทธิภาพสูงสุดด้วย Bun Runtime และการจัดการ UI ที่ซับซ้อนอย่างเป็นระบบ

## 🛠️ Chosen Stack

- **Framework:** Next.js 15 (App Router)
- **Runtime & Package Manager:** [Bun](https://bun.sh/)
- **State Management:** Zustand
- **Table Engine:** TanStack Table v8 (Server-side Mode)
- **Styling:** Tailwind CSS v4 + Shadcn/UI
- **Form Handling:** React Hook Form + Zod

## ⚙️ Setup & Run Instructions

1. ติดตั้ง Dependencies:
   ```bash
   bun install
   ```
2. รัน Development Server:
   ```bash
   bun run dev
   ```
3. เข้าใช้งานที่ http://localhost:3000
